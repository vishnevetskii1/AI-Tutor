01

Что умеет бот

AI-Репетитор — Telegram-бот для пошагового обучения. Он определяет уровень, строит персональный курс по загруженным PDF, объясняет темы, задаёт квизы и сохраняет прогресс по каждой дисциплине отдельно.

Учебный план

Составляет персональный план по реальной структуре PDF-материалов. Количество тем не фиксировано: обычно от 5 до 20.

Учебные блоки

Объясняет каждую тему структурированно, а длинные блоки может разбивать на несколько учебных вопросов со степпером.

Проверочный квиз

После каждого блока — 3 вопроса с выбором ответа. Нужно пройти порог 2 из 3, чтобы двигаться дальше.

XP и ачивки

Зарабатывай опыт за правильные ответы и разблокируй достижения по мере прогресса.

Несколько курсов

Можно одновременно изучать несколько дисциплин и переключаться между ними.

02

Первый запуск

  1. 1 Найди бота в Telegram и нажми Start или отправь команду /start.
  2. 2 Бот поприветствует и покажет список доступных дисциплин в виде кнопок. Нажми на нужную.
  3. 3 Начнётся онбординг — 3 вопроса, чтобы определить твой уровень. Отвечай честно.
  4. 4 Бот построит персональный план по загруженным материалам и покажет его тебе. Число тем зависит от дисциплины.
  5. 5 Нажми «Начать обучение» — и получишь первый учебный блок.
Подсказка: если ты уже учился раньше и хочешь начать заново — используй команду /reset. Прогресс по текущей дисциплине будет сброшен.
03

Онбординг — определение уровня

Бот задаёт 3 вопроса. На основе ответов он определяет твой уровень и адаптирует все объяснения под него.

  1. 1 Самооценка: «Как бы ты оценил свои знания? (1 — новичок, 5 — продвинутый)» — напиши число от 1 до 5 или словами: новичок / средний / продвинутый.
  2. 2 Прошлый опыт: «Изучал ли ты эту тему раньше?» — напиши кратко: не изучал, в институте 2 года, самостоятельно.
  3. 3 Цель: «Какая у тебя цель?» — напиши: экзамен, для работы, для себя, собеседование.
Новичок (1–2)

Объяснения с нуля, бытовые аналогии, медленный темп, каждый термин расшифровывается.

Средний (3)

База известна, упор на понимание связей, переход от смысла к формуле.

Продвинутый (4–5)

Плотный стиль, строгие формулировки, нюансы, сложные случаи и граничные условия.

04

Процесс обучения

Каждый цикл — один блок из плана. Сначала идёт объяснение, затем по желанию можно открыть чат по текущему блоку, после этого пройти квиз. Длинный блок бот может выдавать по частям через кнопку перехода к следующему учебному вопросу.

📋 Показ блока → ➡️ Учебные вопросы → 💬 Вопросы (опционально) → 📝 Квиз → ✅ Пройдено / 🔁 Повтор → ➡️ Следующий блок
  1. 1 Учебный блок — бот присылает развёрнутое объяснение темы с определениями, примерами и подзаголовками. Если блок длинный, он пойдёт несколькими шагами с кнопкой перехода к следующему учебному вопросу.
  2. 2 Режим вопросов — после блока появляются кнопки. Нажми «💬 Спросить по блоку» и пиши вопрос только по текущей теме. Бот ответит как репетитор один на один.
  3. 3 Квиз — нажми «Пройти квиз». Три вопроса подряд, каждый с 4 вариантами. Отвечай кнопками A / B / C / D.
  4. 4 Если 2 или 3 из 3 правильно — блок пройден, начисляется XP, открывается следующая тема. Если нет — бот объяснит блок заново.
Важно: в режиме вопросов бот отвечает только по текущей теме. Если спросить о другом — мягко вернёт к теме. Это сделано специально, чтобы не уходить в сторону.
Ещё один нюанс UX: свободный текст бот принимает в основном во время онбординга и в режиме чата по блоку. В остальных состояниях лучше использовать кнопки под сообщениями.
05

Команды бота

/start Начать или вернуться к выбору дисциплины. Если курс уже идёт — предложит продолжить или выбрать другой.
/progress Показать текущий прогресс: пройденные темы, XP, уровень и текущая дисциплина.
/plan Показать полный учебный план с темами и подтемами. Пройденные блоки отмечены.
/course Открыть список всех курсов: начатые, активные и доступные для старта. Отсюда же можно переключаться между дисциплинами.
/skip Пропустить текущий блок и перейти дальше по плану. Полезно, если тема уже знакома.
/reset Сбросить прогресс по текущей дисциплине и начать заново с онбординга.
/help Показать краткую справку по командам.
Подсказка: кнопки в чате удобнее команд. Используй команды когда нужно быстро переключиться или проверить прогресс.
06

Квиз и система оценки

Квиз — 3 вопроса с множественным выбором, сгенерированные специально по содержанию твоего учебного блока. Вопросы меняются каждый раз при повторном прохождении.

  1. 1 Нажми кнопку «Пройти квиз» под учебным блоком.
  2. 2 Бот задаёт вопросы по одному. Под каждым — кнопки A, B, C, D. Нажимай ответ.
  3. 3 После трёх вопросов бот показывает результат: сколько правильных и сколько XP заработано.
  4. 4 Порог прохождения — 2 из 3 (≥67%). При провале бот объяснит блок заново, потом новый квиз.
Прошёл (≥ 2/3)

Блок помечается как пройденный. Начисляется XP. Открывается следующая тема.

Не прошёл (< 2/3)

Бот показывает блок заново с другим акцентом. Генерирует новые вопросы квиза.

07

XP и достижения

За каждый правильный ответ в квизе начисляется опыт (XP). Посмотреть баланс — /progress.

Результат квиза XP за попытку Примечание
3 из 3 (идеально) +30 XP Даёт ачивку «Идеальный результат»
2 из 3 +20 XP Блок засчитывается как пройденный
1 из 3 +10 XP Блок повторяется
0 из 3 +0 XP Блок повторяется

Достижения разблокируются автоматически при выполнении условий — бот сообщает о них сразу после квиза.

08

Дисциплины и курсы

Доступные дисциплины зависят от того, какие материалы загрузил администратор. Список показывается при /start.

Переключение курсов

Команда /course или кнопка 📚 Все курсы покажет все курсы. Кнопка 📋 Мой план возвращает к плану активной дисциплины.

Независимые прогрессы

У каждой дисциплины свой план, XP и прогресс. Переключение не сбрасывает другие курсы.

09

Частые вопросы

Бот долго не отвечает — что делать?
Генерация учебного блока занимает 10–30 секунд. Если ответа нет больше 2 минут — напиши любое сообщение или нажми кнопку ещё раз.
Можно ли пропустить тему и перейти к следующей?
Да. Для этого есть команда /skip. Она пропускает текущий блок и переводит на следующий. Используй её аккуратно: темы в курсе связаны между собой.
Объяснение непонятное — можно попросить объяснить иначе?
Да. Нажми 💬 Спросить по блоку и напиши, что именно непонятно. Бот ответит в режиме репетитора. Если квиз не пройден, при повторе блок тоже будет объяснён с другим акцентом.
Как начать заново не теряя другие курсы?
Команда /reset сбрасывает только активный курс. Другие дисциплины не затрагиваются.
Почему бот не отвечает на вопросы не по теме?
QA-режим специально ограничен текущей темой и подтемами блока. Это помогает оставаться в фокусе. Для другой дисциплины или другого блока открой /course или кнопку 📚 Все курсы и переключись.
Сохраняется ли прогресс если закрыть Telegram?
Да. Весь прогресс сохраняется в базе данных на сервере автоматически после каждого шага. Можно продолжить в любое время.
10

Для администратора

Базовые операции администратора доступны через Telegram. Расширенные — через консоль сервера.

/admin Открыть панель администратора (только для пользователей из списка admin_user_ids).
Режим обслуживания Включить через панель /admin. Пользователи видят сообщение «Бот на техническом обслуживании».
Статистика Список пользователей, их прогресс по курсам, XP и текущие состояния — в панели /admin.
Документация Из панели /admin можно отправить в чат актуальные HTML-файлы: архитектуру, историю разработки и руководство пользователя.
Доступ Администратор может включать техработы и отдельно отключать конкретных пользователей через admin-панель.
Запуск бота: docker compose up -d — поднимает бот + Qdrant. Логи: docker compose logs -f bot. Переменные окружения в .env (TELEGRAM_BOT_TOKEN, OPENROUTER_API_KEY, LLM_PROVIDER).
